The Staffing Model:

The entire staffing model is built upon the output of one highly productive coordinator. This is the "biggest lever" in the system.  
‚Ä¢ **The Catalyst:** **1 Coordinator** making 500 manual calls per week is expected to book **200 appointments**.  
‚Ä¢ **The Ripple Effect:** These 200 appointments create the need for a full support team:  
    ‚ó¶ **13 Junior Specialists** (to run the 200 initial discovery meetings).  
    ‚ó¶ **6 Senior Specialists** (to run closing appointments).  
    ‚ó¶ **2 Junior Associates** and **4 Senior Associates** (to build the 100 cases generated from the meetings).  
    ‚ó¶ **2 New Business Service People** (to process the \~240 closed deals per month).
